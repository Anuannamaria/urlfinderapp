Section 1

Overview

The Utility Boundary Research Tool helps researchers and GIS professionals quickly identify the most authoritative publicly available URLs that define utility boundaries or service areas for any utility entity in the United States.

Given a state, county, agency name, and utility type, the tool searches multiple official source categories and returns the top 5 ranked URLs, each labelled with its source priority so you always know how reliable the link is.

Two modes of operation

  • Manual Search — look up a single utility entry by filling in a form.
  • Bulk Upload — upload a CSV or Excel file containing many entries and receive ranked URLs for every row at once.
ℹ️

The tool only surfaces publicly available information from official or government sources. It never synthesises or fabricates URLs — every result links to a real page that was found via live search.

Section 2

Source Priority System

Every result is scored and labelled according to a five-tier source hierarchy. Results from higher-priority sources are more authoritative and should be preferred when multiple options are returned.

Tier Source Type Examples Why it is preferred
Priority 1 Official Utility Website districtname.org, ruralwater.org Primary source published by the utility itself
Priority 2 Official GIS Portal hub.arcgis.com, experience.arcgis.com Interactive GIS portals managed by the agency or county
Priority 3 ArcGIS REST Services services.arcgis.com/…/MapServer Machine-readable GIS layer endpoints; directly downloadable
Priority 4 County / State Government county.ok.gov, state.ok.us Official government pages with boundary references
Priority 5 Official PDF / Ordinance Resolutions, service area maps in PDF Legally adopted documents showing boundary extents
✅

When researching a boundary, always check the Priority 1 result first. Only move to lower-priority results if the higher-priority source does not contain the boundary polygon or service area map you need.

Section 3

Manual Search

Use Manual Search when you need to look up a single utility entry. Navigate to the Manual Search tab at the top of the application.

Step-by-step

  1. Open the application and make sure the Manual Search tab is selected (default).

  2. Fill in the State field — enter the full state name (e.g. Oklahoma).

  3. Fill in County / City / District — enter the county or district name (e.g. Canadian County). This field is optional but improves accuracy.

  4. Fill in Agency / ENC Name — enter the full official name of the utility agency (e.g. Canadian County Rural Water Sewer And Solid Waste Management District 4).

  5. Select the Utility Type from the dropdown — Water, Sewer, Stormwater, Wastewater, Gas, Electric, or Solid Waste.

  6. Click Find URLs. The tool runs six targeted search queries in parallel and scores every result it finds.

  7. Review the Top 5 Results displayed on the right, ordered by relevance and source priority.

Section 4

Bulk Upload

Use Bulk Upload when you have a list of utility entries (for example, the sample task sheet from the research brief). The tool processes every row automatically and lets you download all results in a single Excel file.

Step-by-step

  1. Click the Bulk Upload tab at the top of the application.

  2. Upload your file — drag and drop a .csv, .xlsx, or .xls file onto the upload zone, or click it to browse. The file must contain the columns described in Section 5.

  3. Once uploaded, you will see a table listing all rows with a status indicator for each. Click Start Search.

  4. The tool processes rows one at a time. A spinner next to each row indicates it is being searched. A green tick means results are ready; a red icon means the search encountered an error for that row.

  5. Click any completed row to expand it and preview its top 5 URLs directly in the table.

  6. When all rows are finished, a Download Excel banner appears at the bottom of the screen. Click it to save the complete results file.

✅

You do not need to wait for all rows to finish. The Download Excel button (with a row count badge) is available as soon as the first row completes — use it to grab partial results at any time.

Stopping and re-running

  • Click Stop at any point to pause the search mid-way. Completed rows keep their results.
  • Click Re-run to search all rows again (useful after fixing input data).
  • Click Clear to remove all data and start fresh.

Section 5

Preparing Your CSV / Excel File

Your input file should have column headers in the first row. The tool recognises flexible column name variations, so you do not need to rename your existing spreadsheet columns if they are close matches.

Column Required? Accepted Header Names Example Value
State Required State, State Name Oklahoma
County Optional County, County Name, City, District, City / County / District Name Canadian County
Agency Required Agency, Agency Name, ENC Name, ENC, Utility Name, Name Canadian County Rural Water Sewer And Solid Waste Management District 4
Utility Type Required Utility Type, Utility, Type Water / Sewer / Stormwater / Gas / Electric / Solid Waste
📋

The column name matching is case-insensitive and ignores extra spaces and punctuation. For example, Utility Type, utility_type, and UTILITY TYPE are all recognised as the Utility Type column.

Sample file layout

Accepted file formats

  • .csv — comma-separated values (UTF-8 recommended)
  • .xlsx — Excel workbook (2007 or later)
  • .xls — Excel 97–2003 workbook
⚠️

The tool reads only the first sheet of multi-sheet workbooks. Ensure your data is on Sheet 1.

Section 6

Understanding Results

For each search entry, up to 5 results are returned, sorted from most to least relevant. Each result card shows:

Field Description
Rank Position 1–5, where 1 is the best match overall.
Priority Badge Colour-coded label showing the source tier (P1–P5) per the priority system in Section 2.
Title Page title from the source website as indexed by the search engine.
URL Direct link to the source. Click to open in a new tab and verify the content.
Score Internal relevance score (higher = better). Useful when comparing two results of the same priority tier.
Description Click "Show description" to expand the snippet from the search index for additional context.

How results are scored

The scoring system evaluates each URL on several criteria:

  • Domain type — .gov, .org, or arcgis.com sub-domains carry higher base scores.
  • Agency name match — words from the agency name appearing in the URL path boost the score.
  • State and county match — URLs containing the state or county name score higher.
  • Utility type keywords — URLs containing "water", "sewer", "stormwater", etc. receive a bonus.
  • Boundary language — URLs containing words such as "boundary", "service-area", "district", "gis", or "map" receive a bonus.
  • Domain diversity — the top 5 results are drawn from different domains where possible to give a broader picture.
✅

Always verify a URL before using it as a primary source. Confirm that the jurisdiction matches, the utility type is correct, and the page is published by an official entity — especially for Priority 3–5 results.

Section 7

Downloading & Exporting Results

Manual Search export

After a manual search completes, click the Export to Excel button in the top-right of the results panel to download a single-row Excel file for that entry.

Bulk Upload export

Three ways to download bulk results:

  1. Download button in the toolbar — appears as soon as the first row finishes, with a badge showing how many rows are ready.

  2. Partial download during search — a blue notice appears mid-search. Click Download N rows to save results collected so far without waiting for all rows.

  3. Sticky bottom banner — once every row has been processed, a confirmation banner appears at the bottom of the screen with a prominent Download Excel button.

Output file structure

The exported Excel file (utility_boundary_results.xlsx) contains one row per input entry with the following columns:

Section 8

Sources to Avoid

The scoring system automatically penalises and filters out the following source types. They will not appear in the results:

  • Wikipedia (wikipedia.org)
  • Real estate platforms — Zillow, Realtor.com, Redfin, Trulia, Homes.com
  • Social media — Facebook, Twitter / X, LinkedIn
  • Community Q&A — Reddit, Quora
  • Blogs — Blogspot, WordPress, Medium
  • User-generated maps and unofficial third-party mapping websites

If you manually encounter a URL from one of these sources in a search engine, do not record it as a primary source. Use it only to cross-reference and verify information found from an official source.

Section 9

Tips for Best Results

Use the full official agency name

Enter the complete, official name of the utility agency exactly as it appears in regulatory filings or official documents. Abbreviated names reduce matching accuracy.

Better: Caddo County Rural Water Sewer And Solid Waste Management District 3
Worse: Caddo County Water District 3

Include the county even when it is part of the agency name

Filling in the County field separately (even if the county name also appears in the Agency field) gives the scoring algorithm an additional signal, improving result accuracy for rural districts.

Run separate searches per utility type

If an agency covers both Water and Sewer services, create two separate rows in your input file — one for Water and one for Sewer. This ensures the search targets the correct service area for each utility type.

Verify ArcGIS REST layer URLs

Priority 3 (ArcGIS REST) results ending in /MapServer or /FeatureServer can be opened in ESRI ArcGIS Online or QGIS directly. Copy the URL into ArcGIS Online → Add Layer from Web → ArcGIS Server to preview the boundary layer before downloading.

Check confidence against multiple results

If the first result is a Priority 4 or 5 source, check whether results 2–5 include a higher-priority source. It is worth opening all 5 results to identify the most authoritative one before recording your finding.

Section 10

Frequently Asked Questions

Why are no results returned for some rows?

The search could not find any publicly indexed pages that pass the minimum relevance score and are not from excluded domains. This often occurs for very small or rural utility districts with limited web presence. Try simplifying the agency name or searching the county government website manually.

A result URL is broken or returns a 404 error. What should I do?

URLs are discovered live from search-engine indexes. Index caches can sometimes reference pages that have been moved or removed. Note the domain and search it directly — the content may have moved to a new URL on the same site.

The search is taking a long time for bulk uploads. Is that normal?

Yes. Each row runs six targeted search queries. For large files, processing 100 rows typically takes 5–10 minutes. You can download partial results at any time without waiting for all rows to complete.

Can I re-use the output Excel file as an input file?

No — the output file contains extra columns (URL 1–5, Priority 1–5, etc.) that are not expected in the input format. Keep your original input file separate and use it for re-runs.

How do I record the confidence level in my output?

The Priority label in the exported file serves as the confidence indicator: Priority 1 corresponds to High confidence, Priority 2–3 to Medium–High, Priority 4–5 to Medium–Low. Add a separate "Confidence Level" column to your output spreadsheet based on the priority tier of the URL you select as your final answer.

Is the search tool connected to the internet?

Yes. Searches are performed live against public search engines and scored in real time. An active internet connection is required.
